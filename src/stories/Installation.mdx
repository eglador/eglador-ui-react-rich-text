# Installation

Package installation, peer dependencies, and Tailwind v4 integration.

---

## 1. Install the package

```bash
npm install eglador-ui-react-rich-text
```

```bash
yarn add eglador-ui-react-rich-text
```

```bash
pnpm add eglador-ui-react-rich-text
```

## 3. Bundled Lexical packages

All Lexical packages ship as **direct dependencies** — you don't need to install them separately:

- `lexical` (core)
- `@lexical/react` (React bindings + plugins)
- `@lexical/rich-text` · `@lexical/list` · `@lexical/link` · `@lexical/code` · `@lexical/table`
- `@lexical/markdown` · `@lexical/html` · `@lexical/selection` · `@lexical/utils`

If you need a custom Lexical version, add it as a peer dependency in your own project — npm will resolve to your version.

## 4. Add `@source` for Tailwind v4

Tailwind needs to scan classes inside the package. Add the `@source` directive to your global stylesheet:

```css
@import "tailwindcss";

@source "../node_modules/eglador-ui-react-rich-text";
```

## 5. Server components note (Next.js)

All editor components are marked `"use client"` (Lexical requires the DOM). Render them inside client components:

```tsx
// app/editor/page.tsx
import { MyEditor } from "./editor";

export default function Page() {
  return <MyEditor />;
}
```

```tsx
// app/editor/editor.tsx
"use client";

import {
  RichTextEditor,
  RichTextToolbar,
  RichTextContent,
} from "eglador-ui-react-rich-text";

export function MyEditor() {
  return (
    <RichTextEditor>
      <RichTextToolbar />
      <RichTextContent />
    </RichTextEditor>
  );
}
```
