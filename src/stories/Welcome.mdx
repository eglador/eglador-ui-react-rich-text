A modern, compound-component rich text editor for React — built on **Lexical** and styled with **Tailwind CSS v4**.

---

## Installation

```bash
npm install eglador-ui-react-rich-text
```

## Setup

Add the following to your global stylesheet (e.g. `app/globals.css`):

```css
@import "tailwindcss";
@source "../node_modules/eglador-ui-react-rich-text";
```

## Quick Start

```tsx
import {
  RichTextEditor,
  RichTextToolbar,
  RichTextContent,
  RichTextPageSize,
} from "eglador-ui-react-rich-text";

<RichTextEditor>
  <RichTextToolbar />
  <RichTextContent draggable floatingToolbar />
  <RichTextPageSize />
</RichTextEditor>
```

## Features

- **Compound API** — root + toolbar + content slots, composition over configuration
- **20+ toolbar features** — undo/redo, headings (h1–h6), bold/italic/underline/strikethrough/code, text transform (case + sub/super/highlight), text & background color, lists, alignment (LTR/RTL + indent/outdent), quote, link, insert (HR/page break/table)
- **Notion-style drag handle** — block reorder + (+) insert menu
- **Floating selection toolbar** — Medium/Notion pattern
- **Auto-opening link editor** — cursor enters link → form appears with `url`/`target`/`rel`
- **Table support** — grid picker insert + cell-anchored action menu (insert/delete row/col, merge cells, delete table)
- **Page size simulator** — DevTools-style bottom bar with auto-flip viewport-aware popovers
- **Custom nodes** — official `HorizontalRuleNode` + custom `PageBreakNode` decorator
- **HTML / JSON / Markdown** — round-trip serialization on `onChange` and via `initial*` props
- **TypeScript-first** — full type safety + exported feature unions

## Lexical packages bundled

The following Lexical packages ship as direct dependencies — no separate install:

- `lexical` — core editor
- `@lexical/react` — React bindings
- `@lexical/rich-text` — heading, quote, paragraph
- `@lexical/list` — bullet, numbered, check lists
- `@lexical/link` — links and auto-links
- `@lexical/code` — code blocks
- `@lexical/table` — tables with cell selection + merge
- `@lexical/markdown` — markdown shortcuts + transformers
- `@lexical/html` — HTML import/export
- `@lexical/selection` — selection helpers, color/style patches
- `@lexical/utils` — `mergeRegister` and helpers
