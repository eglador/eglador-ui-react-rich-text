A modern rich text editor library built on **Lexical** and **Tailwind CSS v4** for React-based projects.

> **Status:** Setup complete — components in development. Use this page to verify Storybook configuration. Stories will appear in the sidebar as components are added.

---

## Stack

- **React** 18+ (peer)
- **Lexical** 0.44+ (Meta's extensible editor framework)
- **Tailwind CSS** v4 (peer)
- **TypeScript**
- **tsup** (CJS + ESM bundle)
- **Storybook** 10 (Vite + addon-docs/a11y/vitest)

---

## Lexical packages bundled

The following Lexical packages ship as direct dependencies — you don't need to install them separately:

- `lexical` — core editor
- `@lexical/react` — React bindings
- `@lexical/rich-text` — heading, quote, paragraph
- `@lexical/list` — bullet, numbered, check lists
- `@lexical/link` — links and auto-links
- `@lexical/selection` — selection helpers
- `@lexical/utils` — utility functions

## Roadmap

- `RichTextEditor` — full WYSIWYG with toolbar
- `Toolbar` — formatting controls (bold, italic, headings, lists, links)
- `MarkdownShortcuts` — `**bold**`, `*italic*`, `# heading` etc.
- Custom node plugins (image, table, code block)
- Read-only viewer
- Initial state via JSON / HTML / Markdown
- Output serialization
